**The Tech Stack**

This project is built with a focus on performance, modern React practices, and automated deployment.

### Frontend
- **Framework:** React 
- **Build Tool:** Vite (for lightning-fast builds)
- **Styling:** CSS
- **Routing:** React Router

### Infrastructure & DevOps
- **Host:** AWS Lightsail (Linux VPS)
- **Web Server:** Caddy (Automatic HTTPS & Reverse Proxy)
- **CI/CD:** GitHub Actions (Automated build & deploy on push to `main`)
- **Package Manager:** npm
